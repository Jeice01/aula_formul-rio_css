O que é um Formulário?

  • Forms tem o papael de receber dados do usuário e enviar para um servidor;
  • Podemos validar dados;
  • Tags mais utilizadas são: form, label e input;
  • A tag form cria o formulário e o delimita;
  • Label descreve os inputs
  • Os inputs é a tag que inserims dados, temos vários tipos: number, email, text e entre outros.

Para criar um formulário em html

  • Para criar um formulário vamos precisar da tag form, que encapsula todos os elementos do formulário;
  • Dentro da tag form, temos labels e inputs, mas podemos usar outras tag como divs;
  • A tag input tem um atributo chamado type, que é onde definimos o propósito do input;
  • Ex.: o input do tipo "text", recebe dados do tipo texto.

Temos outras tags como os select(options), checkbox, radio, datalist, fieldset(legend) e entre outras.
